Rubik Cube Solver

A Rubik's Cube solver built with C++ uses a combination of IDDFS, BFS, and DFS algorithms with a heuristic approach. The project implements Object-Oriented Programming (OOP) principles to structure the solution, efficiently solving the cube by iterating through possible states. The IDDFS algorithm combines depth-first and breadth-first search techniques to explore the solution space, ensuring an optimal and systematic approach to solving the Rubik's Cube.
